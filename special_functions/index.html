[−][src]Crate special_functions

Library providing pure rust implementation of various special functions, with particular focus to high-energy particle physics.

This library will attempt to provide full 64-bit precision by default. In circumstances where such precision may be particularly slow to achieve, a less accurate implementation may be provided.

This library is still undergoing development.
